Add the `create-chat-sdk` CLI for scaffolding webhook-only Next.js Chat SDK bot projects. Supports interactive prompts, non-interactive `--adapter` selection from the `chat/adapters` catalog, coding-agent detection with an `--interactive` escape hatch, and generated `src/lib/bot.ts`, `.env.example`, `next.config.ts`, and README output per selected adapter.
